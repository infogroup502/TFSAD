# TFSAD

## Requirements
The recommended requirements for LFTSAD are specified as follows:
- arch==6.1.0
- einops==0.6.1
- matplotlib==3.7.0
- numpy==1.23.5
- pandas==1.5.3
- Pillow==9.4.0
- scikit_learn==1.2.2
- scipy==1.8.1
- statsmodels==0.14.0
- torch==1.13.0
- tqdm==4.65.0
- tsfresh==0.20.1

## Usage
1. Install Python 3.6, PyTorch >= 1.4.0
2. Download the datasets
3. To train and evaluate TFSAD on a dataset, run the following command:
```bash
python main.py 
```
